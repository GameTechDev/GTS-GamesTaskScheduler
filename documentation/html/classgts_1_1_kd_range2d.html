Detailed Description

template<typename XIterType, typename YIterType = XIterType>
class gts::KdRange2d< XIterType, YIterType >

An iteration range over a 2D data set. The range is divided recursively along the largest dimension until a minimum size is reached for all dimensions. The result is a Kd-tree were each leaf represents a unit of work.
