Detailed Description

template<typename XIterType, typename YIterType = XIterType, typename ZIterType = XIterType>
class gts::OctRange< XIterType, YIterType, ZIterType >

An iteration range over a 3D data set. The range is divided recursively along in each dimension until a minimum size is reached for all dimensions. The ideal result is a oct-tree were each leaf represents a unit of work. Non-cube ranges may not have complete trees.
