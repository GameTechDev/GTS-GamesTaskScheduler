Detailed Description

template<typename T, typename TAllocator = gts::AlignedAllocator<GTS_NO_SHARING_CACHE_LINE_SIZE>>
class gts::QueueSPSC< T, TAllocator >

A single-producer, single-consumer queue. Properties:

  • Wait-free.
  • Unbound.
  • Linearizable.
