Detailed Description

template<typename XIterType, typename YIterType = XIterType>
class gts::QuadRange< XIterType, YIterType >

An iteration range over a 2D data set. The range is divided recursively along in each dimension until a minimum size is reached for all dimensions. The ideal result is a quad-tree were each leaf represents a unit of work. Non-square ranges may not have complete trees.
